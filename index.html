Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: stages all changes for commit
  • git commit -m: commits staged changes with messaged following "-m"
  • git pull origin main: pulls latest from main branch
  • git push origin feature/[branch name]: pushes committed branch to GitHub
